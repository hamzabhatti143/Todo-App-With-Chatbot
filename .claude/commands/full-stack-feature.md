# Full-Stack Feature Workflow

## Steps
1. /specify feature in specs/features/
2. /plan for frontend and backend
3. /tasks for both layers
4. Implement backend (use @fastapi-backend-dev)
5. Implement frontend (use @nextjs-frontend-dev)
6. @api-integration-specialist verify integration
7. @code-reviewer review both layers
8. @documentation-writer update docs

## Example Usage
/specify Task CRUD feature
@fastapi-backend-dev implement backend
@nextjs-frontend-dev implement frontend
@api-integration-specialist test integration
@code-reviewer review
@documentation-writer update docs

Use for all features requiring both frontend and backend.
